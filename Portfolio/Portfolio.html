Quiz App

The Quiz App, built with React Native, offers users a straightforward and engaging platform to take quizzes. It incorporates a Question component that supports various answer types, including true/false, multiple choice, and multiple answers.

Features:

  • Questionnaire: Supports true/false, multiple-choice, and multiple-answer formats.
  • Results: Provides quiz results.

Encountered Challenge: One challenge was handling multiple answers in the multiple-choice questions. This required separating the functionality to ensure correct behavior. Additionally, there were issues with including true/false responses in the results, which needed resolution.

Crafted in Figma, the Flash Card Design offers an intuitive layout for effective learning.
